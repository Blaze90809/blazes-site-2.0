Blaze's Work

Here are a few of the personal projects I have spent my time developing.

Percent Back Application

This Percent Back application gives cross-country skiers a program to help track and anaylize results. The application takes information from completed races and calculates percent-back from the leader. Users have the opportunity to view results in a table. This program was written with React, Express, Node.js and MongoDB.

 Project Footprint

The Footprint environmental impact calculator is a full-stack application that gives organizations the opportunity to upload their data and view a timeline-based breakdown of carbon emissions. Project Footprint was a group project created by Blaze and Prime students Zack Stout, Jaffa Aharanov and Holly Tuhake.

 Kiosk App

This Kiosk application is written to match a touch-screen kiosk that allows customers to purchase items. The full-stack application stores products in a Mongo database and utilizes GET routes to display items by category. Items and total cost are also displayed in a receipt on the side. The program is written using the MEAN stack - Mongo, Express, AngularJS, and Node.js.
